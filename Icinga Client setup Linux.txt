#Monitor Linux OS using Icinga

##Icinga client(windows and Linux)

step 1, config Icinga2 master
	Open firewall for the port 5665
	run node wizard
	
step 2, Install and config icinga2 client
	open firewall for the port 5665
	install icinga2 and nagios plugins
	run node wizard
	update the config from client to master
	

Step 1: Master side
#Stop firewall
#Run node wizard using following command
icinga2 node wizard
Specify master setup : No for master Yes - client
specify common name : icinga02
bind host -- Leave Empty


Step 2: Client step
# Install icinga02 client
yum install icinga2
yum install nagios-plugins-all
icinga2 node wizard
#give all the required info
#generate key using Hint at end
#restart icinga2 service to apply changes
service icinga2 restart


#go to Master node and enter following command to check nodes list

icinga2 node list
icinga2 node update-config
service icinga2 restart

#click on check now from webURL
